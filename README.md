신한금융그룹 빅데이터 플랫폼 - 관리자 페이지

[수행기간]
2018.07 - 2019.01

[수행 내역]
- Spring Security을 Customize하여 로그인(SHA-256) 및 권한 관리 체계 구현
- jsp / ajax / jQuery
- 발주사의 요구사항에 맞춰 일부 Customize

- 수주사에서 제공한 Rule Set과 Sonacube를 이용한 Code Inspection
- 보안팀의 점검결과 보고서를 토대로 보안성검토 조치
